## Decisions Made

- **Phantom spacer over padding-right** — `::after { display: inline-block; width: 65px }` on `textContentItem` reserves timestamp space only on the last text line; all other lines use full width. Works because `MarkdownWrapper` uses `forceInline: true` (confirmed by reading source).
- **Module-level Map for cache** — chosen over React Context or sessionStorage; no serialization overhead, survives re-renders, simple to evict (Map insertion order = FIFO).
- **`maxWidth: 260` on bubbles** — image (250px) + bubble padding (5px × 2) = exactly 260px; image fills bubble with no horizontal gap.
- **`RESTORE_CACHE` vs reusing `LOAD_MESSAGES`** — separate action to cleanly replace state; `LOAD_MESSAGES` still handles dedup when background fetch returns.
- **emoji-mart `theme` prop** — simplest fix; `emoji-mart` v3 handles all dark styles internally via the `theme` prop; no CSS overrides needed.
